# PNG Exports

This directory contains PNG exports of the Mycelium logo in various sizes and formats for different use cases.

## Files

- `logo.png` - Primary logo for documentation (512x512px)
- `logo-small.png` - Small version for inline use (64x64px)
- `logo-large.png` - Large version for headers (1024x1024px)
- `favicon-32.png` - Favicon size (32x32px)
- `favicon-16.png` - Small favicon (16x16px)

## Usage

The primary `logo.png` file is optimized for documentation and general web use at 512x512 pixels with transparent background.
