# Mycelium Logo Assets

This directory contains all logo design assets for the Mycelium project.

## Structure

- `sketches/` - Initial concept sketches and iterations
- `primary/` - Primary logo variations (horizontal, vertical)
- `icon/` - Icon-only versions
- `wordmark/` - Typography-focused versions
- `exports/` - Final exported formats (PNG, PDF, ICO)
- `guidelines/` - Brand guidelines and usage documentation

## Design System

The logo is built on a 24x24 unit grid system with:
- 2-unit consistent stroke weights
- 1-unit corner radius for subtle softness
- Organic mycelium network patterns balanced with geometric precision
